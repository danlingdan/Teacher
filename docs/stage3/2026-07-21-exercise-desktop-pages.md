# v0.3.0 教师题库与学生闯关练习页面

## 1. 教师题库页面

顶部导航新增“教师题库”，页面接入真实 `ExerciseManagementService`，支持：

- 查看全部启用/停用题目及版本。
- 新建和编辑标题、说明、知识点、难度、数据集、参考 SQL、评测规则与最多三级提示。
- 配置列、行和值、顺序、指定行数及必需 SQL 结构规则。
- 复制题目；副本默认停用。
- 启用或停用题目，并使用版本号防止覆盖较新的修改。
- 导入版本化 JSON 题包。
- 导出选中题目；未选中时导出全部题目及其依赖数据集。

参考 SQL、初始化 SQL 和评测规则只出现在教师页面及基础设施层，不进入学生 DTO。

## 2. 学生闯关练习页面

顶部导航新增“闯关练习”，页面接入真实 `ExerciseCatalogService` 与 `ExercisePracticeService`，支持：

- 浏览 20 道启用题目，查看题目说明、知识点和难度。
- 开始题目并创建隔离数据集。
- 编写并运行单条 `SELECT`，动态显示结果列和数据行。
- 提交确定性评测，显示总体结果和逐项反馈。
- 逐级请求提示，不展示参考 SQL 或正确结果。
- 重置当前隔离数据集。
- 答对后关闭会话并锁定操作，需重新开始才能继续。

## 3. 线程与错误处理

- 题库 JDBC、题包文件读写、会话初始化、SQL 运行和评测全部进入共享后台执行器。
- JavaFX Application Thread 只负责读取表单与更新控件。
- 耗时操作复用全局 Loading 遮罩。
- 异常通过统一 `ApplicationExceptionMapper` 转换为用户消息，不在页面展示栈信息、数据库路径或参考答案。
- 页面懒加载并由主窗口缓存，切换导航不会丢失尚未提交的编辑器内容。

## 4. 低分辨率适配

两页均使用可调整 `SplitPane`。教师表单放入纵向滚动容器；导航按钮缩小横向留白，保证新增入口在最小 960×600 窗口仍可访问。
