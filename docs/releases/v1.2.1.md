# SQLTeacher v1.2.1

SQLTeacher v1.2.1 是 v1.2.0 的 Windows 发布修正版。

## 修复

- 正式 app-image、EXE 和 ZIP 现在固定写入 `https://api.sqlteacher.tech`，不再回退到不可路由的安全占位域名。
- 打包过程会检查生成的 `SQLTeacher.cfg`；若缺少生产云端地址则立即失败，不生成错误发布包。
- `CloudBaseUrl` 打包参数只接受绝对 HTTPS URL，避免正式版本意外连接明文云服务。

## 发布自动化

- 新增 GitHub Actions Windows 发布流程。
- 推送与 `pom.xml` 版本一致的 `vX.Y.Z` 标签后，自动运行全部 Maven 测试并生成 EXE、便携 ZIP 和 SHA-256 清单。
- Release 在资产全部上传前保持草稿，上传完成后才正式公开。

安装包目前仍未代码签名，下载后请核对 Release 中的 `SHA256SUMS.txt`。
