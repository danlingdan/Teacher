# SQLTeacher v0.4.0

`v0.4.0` 面向教师分析与本地课程资料检索，继续保持 SQLite 核心练习离线可用。

## 主要更新

- 新增教师学情看板，展示会话、尝试、通过率、完成率、平均尝试次数和提交耗时。
- 支持按日期、题目、知识点和错误类型组合筛选。
- 增加题目完成情况、常见错误和知识点薄弱项统计。
- 支持 UTF-8 CSV 导出，并记录筛选边界、生成时间和统计口径。
- 支持 UTF-8 文本与 Markdown 课程资料导入、切片和 SQLite FTS5 本地检索。
- 检索结果展示标题、来源文件和命中片段；删除文档时同步清理索引。
- 增加本机学习数据清理入口，保留题库、连接配置和知识文档。
- 学习事件覆盖练习尝试、通过、失败、提示和知识检索，不记录 SQL、查询或文档原文。

## 下载与运行

下载 `SQLTeacher-0.4.0-windows-x64.zip`，解压后运行 `SQLTeacher\SQLTeacher.exe`。

运行环境：Windows 10/11 x64。发布包包含 Java 运行时，无需单独安装 JDK。

## 已知限制

- 当前发布包仅面向 Windows x64。
- 知识库仅支持不超过 2 MiB 的 UTF-8 `.txt`、`.md` 和 `.markdown` 文件。
- 知识检索为本地 SQLite FTS5 关键词检索，暂不包含向量检索或在线同步。
- 当前为单机统计，不包含账号、班级或多学生身份体系。
- Ollama 与本地模型为可选组件；不可用时核心练习、看板与知识检索仍可运行。
