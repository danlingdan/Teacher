# SQLTeacher v1.1.0

SQLTeacher v1.1.0 在 v1.0 本地优先教学闭环上增加可选云端协作与用户自带 API 的网络 AI。

## 新增功能

- 强制启动身份入口，支持云端登录、学生注册和不上传账号数据的访客模式。
- 访客、学生、教师、管理员四级桌面权限；导航和班级操作按身份动态显示并在控制器再次校验。
- 邮箱注册/登录、短期访问令牌、退出撤销和管理员/教师/学生权限边界。
- 多班级、多教师与学生成员管理，并支持教师发布任务、班级成员查看任务；服务端按班级成员关系授权。
- 学习事件幂等上传、版本增量下载和本地离线保留。
- “云端教学”桌面页，集中完成账号、班级、成员、任务发布、同步和网络 AI 配置。
- OpenAI Chat Completions compatible Provider；API Key 只驻留当前进程内存，不同步、不备份、不写日志。
- 云端 API systemd、Nginx、独立运行账户和部署文档。

## 安全边界

- AI 输出仍须经过 JSON 解析、方言校验、SQL 风险分析和用户预览，不能直接执行 SQL。
- 云端不接收数据库密码、网络 AI API Key 或本地数据库文件。
- 本地学习事件带账号所有者标记；访客记录和其他账号记录不会进入当前账号同步，每个账号使用独立增量游标。
- 学生不能创建班级；撤销后的令牌不能继续访问；重复同步事件不会产生重复记录。
- 云端与网络 AI 不可用时，本地 SQLite 练习、题目评测、统计和知识检索仍可使用。

## 已知限制

- 当前 ECS 测试入口使用裸 IP HTTP，只适合短期联调；正式教学数据必须等待域名与 HTTPS 后使用。
- 访问令牌仅保存在桌面进程内存，应用重启后需要重新登录。
- 网络 AI 目前支持 OpenAI Chat Completions compatible HTTPS 接口；不同厂商的专用协议需另加适配器。
