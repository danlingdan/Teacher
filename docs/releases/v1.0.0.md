# SQLTeacher v1.0.0

SQLTeacher 首个正式单机桌面版本，覆盖安全 SQL 练习、受限外部数据库、确定性题目评测、学情分析、本地知识检索和 Windows 自包含交付。

## 新增与完善

- Windows EXE 安装器、便携 ZIP、统一应用图标和可追溯版本信息。
- `%LOCALAPPDATA%\SQLTeacher` 用户数据目录与旧 `app-data` 自动迁移。
- 升级前自动备份、手动备份、SQLite 完整性校验恢复、失败回滚和演示库复原。
- “设置与数据”双页、`Ctrl+1` 至 `Ctrl+9` 快捷导航和窄屏横向导航。
- 发布性能验证程序和安装契约测试。

## 数据与兼容性

- Maven 版本：`1.0.0`。
- 应用数据库 schema：`4`，兼容 v0.4.0；更早 schema 按顺序迁移。
- 卸载默认保留 `%LOCALAPPDATA%\SQLTeacher`。
- v0.4.0 的题库、事件、连接配置、知识索引和模型选择可继续使用。

## 已知限制

- 安装器未代码签名，Windows 可能提示未知发布者。
- 本轮在 Windows 11 实机完成安装包与 UI 验证；Windows 10 需在独立干净环境补充兼容性抽查。
- MySQL 9.7.1 真实只读账户流程已在 v0.2 阶段通过；MariaDB 由驱动路由和自动化回归覆盖，未在本轮独立服务器复测。
- AI 质量依赖本地 Ollama 模型；无模型时安全降级，不影响核心 SQLite 功能。
