# SQLTeacher v1.2.0

发布日期：2026-07-22

SQLTeacher v1.2.0 将 v1.1 的云端联调能力提升为经正式域名和 HTTPS 保护的教学协作服务，并继续保持本地优先与 SQL/AI 安全边界。

## 新增与改进

- 正式服务地址 `https://api.sqlteacher.tech`，启用 HTTPS 跳转、HSTS、最小安全响应头和本机反向代理边界。
- 8 小时访问令牌与 30 天可撤销刷新令牌；刷新令牌每次使用后轮换，重放旧令牌会被拒绝。
- Windows 使用当前用户 DPAPI 加密保存会话；应用重启可恢复或刷新会话，退出会清除本机密文并撤销服务端令牌。
- 学习事件同步按账号隔离，使用稳定幂等 ID、增量游标、三次有限重试、退避与持久化诊断状态；失败不删除本地记录。
- 任务支持发布、编辑、截止、撤回和归档，过期任务自动截止，归档后不可修改。
- 教师与管理员可查看授权班级摘要并导出 UTF-8 CSV；导出实施服务端授权、审计和电子表格公式注入防护。
- 新增云数据库在线备份、完整性校验、30 天保留和受控恢复脚本。

## 安全边界

- 非回环 HTTP 云服务地址会被客户端拒绝；本机 HTTP 仅用于自动化和集成测试。
- AI 仍只能生成草案，不能访问 JDBC 或直接执行 SQL；所有草案继续经过 Java 侧风险分析。
- 云端不接收数据库密码、网络 AI API Key、本地数据库文件或访客学习记录。
- 学生只能看到所属班级已发布/已截止任务；统计、导出和任务变更在服务端再次验证管理权限。

## 已知限制

- Windows 安装包尚未代码签名，首次运行可能显示来源提示，请核对发布页 SHA-256。
- 会话持久化依赖 Windows DPAPI；其他操作系统会安全降级为进程内会话。
- 云端与网络 AI 是可选增强；不可用时本地 SQLite 练习、确定性评测、统计和知识检索仍可使用。
- 当前证书有效至 2026-10-20；HTTP-01 续期 dry-run 因公网多点验证链路返回 403 未通过，需在 2026-09-20 前配置阿里云 DNS-01 或修复公网验证链路并复测。Certbot 定时器保持启用。
