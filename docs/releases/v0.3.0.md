# SQLTeacher v0.3.0

`v0.3.0` 将 demo 扩展为可用于单机数据库教学的练习闭环。

## 主要更新

- 提供 20 道内置 SQL 练习，并支持练习目录持久化。
- 支持 JSON 练习包导入与导出。
- 每次练习使用隔离的 SQLite 会话，正常退出或异常中断后自动清理。
- 增加确定性 SQL 判题、提示、重置和作答记录。
- 增加教师练习管理页、学生练习页和基础学习进度页。
- 保留 SQL 风险检测、AI 草案安全校验和本地 Ollama 降级能力。

## 下载与运行

下载 `SQLTeacher-0.3.0-windows-x64.zip`，解压后运行 `SQLTeacher\SQLTeacher.exe`。

运行环境：Windows 10/11 x64。发布包包含 Java 运行时，无需单独安装 JDK。

## 已知限制

- 当前发布包仅面向 Windows x64。
- Ollama 与本地模型为可选组件；不可用时 AI 功能会安全降级。
- MySQL 驱动已包含，但练习隔离与判题流程当前以 SQLite 为主。
