## 安全审查

- AI 不能直接执行 SQL 或取得 JDBC 连接；草案必须经过 Java 风险分析。
- 外部数据库默认只读，密码只驻留进程内存，不进入配置、日志、备份元数据或导出。
- 备份 ID 严格校验，ZIP 只接受固定数据库条目，恢复前执行 `pragma integrity_check`。
- 用户数据与安装目录分离，卸载不会误删题库、学习数据或知识文档。
- 首轮安装测试发现 jpackage 默认目录与数据目录重合；已改为 `%LOCALAPPDATA%\SQLTeacher-App`，从保留的旧 `app-data` 恢复并核对数据库 SHA-256 后重新验证。
