## 已知限制

- 当前仅提供 Windows x64 app-image ZIP，不是 MSI/EXE 安装器。
- 学情数据为本机单用户口径，不包含账号、班级和多学生身份。
- 知识检索为 SQLite FTS5 关键词检索，不包含向量检索、在线同步或自动注入 AI Prompt。
- 知识文件限 2 MiB 且仅支持 UTF-8 文本和 Markdown。
- Ollama 当前不可用，但不影响 SQL 练习、确定性评测、教师看板和本地知识检索。
