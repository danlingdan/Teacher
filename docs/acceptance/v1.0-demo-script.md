# SQLTeacher v1.0 十分钟演示脚本

1. 0:00–1:00：启动安装版，展示新图标、首页与 `Ctrl+1` 键盘导航。
2. 1:00–2:30：在内置 demo 执行 `select * from student`，展示结果上限和表结构。
3. 2:30–3:30：输入多语句或禁止语句，展示 Java 侧拦截。
4. 3:30–5:30：完成一道闯关题，运行、提示、提交并说明确定性判题。
5. 5:30–6:30：Ollama 可用时生成 SQL 草案；不可用时展示安全降级。
6. 6:30–7:30：导入课程 Markdown，搜索 `GROUP BY` 并展示来源。
7. 7:30–8:30：打开教师看板，应用筛选并导出 CSV。
8. 8:30–9:30：打开数据库连接，说明 MySQL 只读账号与临时密码策略。
9. 9:30–10:00：打开“版本与数据”，创建手动备份并说明升级、恢复和卸载保留策略。
