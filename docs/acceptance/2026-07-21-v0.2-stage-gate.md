## 1. 验收结论

`v0.2.0` 工程门禁条件通过，但暂不宣布正式发布。

连接管理、SQLite 回归、MySQL 9.7.1 真实只读权限、失败提示、凭据保护、SQL 安全拦截和 Windows app-image 均通过验收。当前唯一发布条件是：原计划明确要求 MySQL 8.x 或兼容 MariaDB 环境，本机只有 MySQL 9.7.1，且没有 MariaDB 实例。

该缺口不否定已完成代码，但在补充目标版本真实兼容验证前，不将 `v0.2.0` 标记为正式可发布版本，也不删除此限制。

## 9. 发布阻塞与解除条件

当前发布阻塞项只有一个：缺少计划指定的 MySQL 8.x 或 MariaDB 真实实例验证。

解除条件：

1. 在独立 MySQL 8.x 或 MariaDB 环境创建隔离只读账号。
2. 复用查询、当前 catalog 元数据、写入拒绝、系统表拒绝和清理矩阵。
3. 验证对应方言驱动和错误分类。
4. 将服务器版本、命令、结果与清理状态写入本记录。
5. 再次运行受影响的增量测试；若期间累计多个功能，再运行全量测试。
