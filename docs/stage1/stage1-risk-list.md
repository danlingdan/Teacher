# 阶段 1 风险清单

| 风险 | 影响 | 当前状态 | 应对措施 |
|---|---|---|---|
| Spring DI 与 JavaFX 生命周期耦合 | 应用退出时资源未释放 | JavaFX `stop()` 中关闭 Spring Context | 后续引入 Controller 工厂时继续保持统一关闭 |
| 运行期数据被误提交 | 仓库污染、泄露本地数据 | `app-data/` 已加入 `.gitignore` | 后续真实数据库、日志和导出文件继续保持忽略 |
| Ollama 服务有但无模型 | AI 功能仍不可用 | HTTP 服务可访问，模型数为 0 | 阶段 3 前安装并记录默认模型 |
| SQLite 初始化结构过早固化 | 后续迁移成本 | 当前只建最小 `app_event` 和 `student` 表 | 阶段 2 设计 demo.db 初始化脚本和恢复机制 |
| UI 页面仍是占位 | 不能支持完整业务流 | 当前只展示状态 | 阶段 2 开始补 SQL 练习页 |
