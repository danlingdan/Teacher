# v0.4 教师看板与课程知识桌面页

## 教师学情看板

- 支持按本机日期、题目、知识点和错误类型组合筛选。
- 展示会话、尝试、提交通过率、题目完成率、平均尝试次数和平均提交耗时。
- 分页展示题目完成情况、常见错误和知识点薄弱项。
- CSV 导出与页面调用同一统计服务，包含筛选边界、UTC 生成时间与口径说明。
- 提供学习数据清理确认，只删除会话、尝试和学习事件。

## 课程知识页

- 支持选择本机 UTF-8 文本或 Markdown 文档导入。
- 文档列表显示标题、来源文件名、切片数和导入时间。
- 检索结果显示来源、标题和带命中标记的片段。
- 删除前明确确认，完成后同时清空对应检索结果。

## 线程与状态

- 统计、CSV 生成/写入、文件读取、索引和检索均在共享后台执行器运行。
- 页面提供加载、成功、空数据与失败状态，不展示原始堆栈。

## 当前验证

- `mvn -q -DskipTests compile`：通过。
- 自动化和真实 JavaFX 走查将在阶段功能完成后集中执行。
