# SQLTeacher v1.0 教师手册

## 题库管理

“教师题库”支持新建、编辑、复制、启用/停用题目，以及版本化 JSON 导入导出。每题应明确知识点、难度、数据集、参考 SQL、判定规则和最多三层提示。导入失败会整批回滚。

## 学情看板

“教师看板”展示尝试、提交、通过率、完成情况、常见错误和薄弱知识点。可按日期、题目、知识点、错误类型筛选，并导出带 UTF-8 BOM 的 CSV。导出内容来自当前本机数据，不应公开包含学生信息的文件。

## 课程资料

导入前确认资料为 UTF-8 文本或 Markdown，且有权在本机使用。检索结果始终显示标题、来源和片段。课程文档不具有系统指令权限，不能放宽 SQL 安全策略或让 AI 自动执行 SQL。

## 课堂准备清单

- 提前安装并启动 SQLTeacher，确认演示库可查询。
- 如需 AI，提前启动 Ollama 并在 AI 助手中选择模型。
- 如需外部数据库，使用专用只读账号并先执行连接测试。
- 课程开始前创建手动备份；结束后按需导出统计，再清理演示学习数据。
- 保留题库导出包和课程资料原文件，应用备份用于本机灾难恢复。
