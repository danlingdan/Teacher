## 2. 软件用途和适用对象

SQLTeacher 是一款面向数据库课程教学、实验练习和课后辅导的 Java 桌面软件。
软件为学生提供内置 SQLite 演示数据库、表结构查看、SQL 编写与执行、查询结果展示、
SQL 风险提示以及本地 AI 辅助生成 SQL 草案等能力；教师可利用执行事件记录了解练习过程。

主要适用对象：

- 高等院校和职业院校数据库课程学生。
- 数据库课程教师和实验指导人员。
- 需要在本地环境练习基础 SQL 的学习者。

## 3. 主要功能

1. 初始化本地 SQLite 应用数据库和演示数据库。
2. 展示数据库表及字段结构。
3. 接收 SQL 输入，执行受控查询并展示列、数据行、耗时和错误状态。
4. 对 SELECT、数据修改、DDL、多语句和禁止语句进行风险分类。
5. 对需要确认的 SQL 显示明确的风险确认信息。
6. 调用本地 Ollama，根据自然语言生成结构化 SQL 草案和教学解释。
7. 对 AI 输出执行 Java 侧解析、字段校验和 SQL 风险检查，AI 不直接执行 SQL。
8. 在 Ollama 不可用、超时或响应异常时提供可理解的降级提示。
9. 记录 SQL 执行、风险拦截和 AI 生成等学习事件。
10. 通过 JavaFX 桌面界面提供 SQL 练习页、表结构页和全局加载状态。

## 4. 技术特点

软件采用 Java 21 和 JavaFX 构建桌面界面，以 Spring Context 组织服务依赖，使用 JDBC
访问 SQLite 数据库，并通过 Jackson 解析本地 Ollama 的结构化 JSON 输出。业务流程按应用、
领域、基础设施和桌面界面分层，SQL 执行统一进入风险分析路径；多语句和禁止语句默认拦截，
高风险操作要求用户确认，AI 生成内容仅作为未受信任的草案展示。
