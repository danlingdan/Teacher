## 2. 官方材料要求摘要

依据《计算机软件著作权登记办法》和政务服务公开指南，通常需要：

1. 按要求填写的软件著作权登记申请表。
2. 软件鉴别材料，包括程序和任一种文档的鉴别材料。
3. 申请人身份证明及必要的权属证明文件。
4. 程序和文档通常各提交前、后连续 30 页；不足 60 页时提交全部。
5. 除特定情况外，程序每页不少于 50 行，文档每页不少于 30 行。
6. 申请文件使用中文和 A4 纸张；软件名称、版本号、申请人署名应在全部材料中一致。

核对来源：

- 北京市政务服务网《计算机软件著作权登记初审》：
  https://banshi.beijing.gov.cn/pubtask/task/1/110000000000/3e283672-76be-4c8c-98e8-0bebe9bd06bf.html
- 《计算机软件著作权登记办法》（国家版权局令第 1 号）：
  https://www.sz.gov.cn/cn/xxgk/zfxxgj/zcfg/content/post_8965812.html

正式递交前应以中国版权保护中心届时的在线申请表、办事指南和系统校验结果为准。

## 3. 下一步顺序

1. 申请人确认 `04-权属与待确认事项.md` 中的关键事项。
2. 冻结用于申请的版本和代码提交，统一软件全称、简称、版本号及署名。
3. 将 `02-软件说明书初稿.md` 扩写并配入最终软件截图，形成 A4 版说明书。
4. 按 `03-源程序鉴别材料方案.md` 生成连续源程序，排版并核对前后各 30 页。
5. 将申请表、说明书、源程序和权属证明逐项交叉检查后再提交。

当前已确认：本软件按多人共同著作权准备，学校为第一著作权人/第一署名单位。学校法定全称、
其余共同著作权人名单与排序、代表申请授权及盖章流程仍待补全。
